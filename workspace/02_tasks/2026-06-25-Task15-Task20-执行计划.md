# Task 15-Task 20 执行计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在现有 CPU 闭环基础上补齐后端探测、GPU 适配、主线程渲染、缓存与性能打点、关键接口回归，以及样例联调记录。

**Architecture:** 继续复用现有 `Worker -> task-runner -> run-frame` 主链路，不重做调度协议；新增能力优先做成可插拔的小模块，再把最少的接线补回 `worker-entry / worker-client / interfaces`。GPU 相关先保证选择逻辑、调用路径和 CPU 一致性，未校准或未实测的点写入联调记录。

**Tech Stack:** TypeScript, Vitest, typed arrays, existing rain iso worker protocol.
